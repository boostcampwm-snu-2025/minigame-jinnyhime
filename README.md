# 🦖 Personal Dino Game (개인화 공룡 미니게임)

사용자의 얼굴 사진을 캐릭터에 입혀서 즐기는,  
구글 공룡게임 스타일의 **간단하고 귀여운 미니게임**입니다! ^_^

## Features
- 얼굴 업로드 & 자동 합성
- 점프/중력 적용 러너 게임
- 선인장 장애물 & 충돌 판정
- 랜덤 크기/간격 장애물 생성
- 게임 오버 이후 얼굴 상태 유지
- 전체 화면 레이아웃

## Tech Stack
- React (Vite)
- CSS (Custom)
- Canvas-Free DOM Animation
